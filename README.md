# Web Project (Football Reservation System)

## 1) Cloning the Repository 

```bash
git clone --recursive https://github.com/nourabouzeid/Football-Match-Reservation.git
```

If you already cloned the repository without the --recursive flag, initialize the submodules manually:
```bash
git submodule update --init --recursive
```

## 2) Setting up the frontend
```bash
cd frontend
npm install
npm start
```
## 3) Setting up the backend
```bash
cd backend
npm install
node index.js
```
